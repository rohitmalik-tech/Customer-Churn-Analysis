# 📊 Customer Churn Analysis Dashboard

## 📌 Project Overview

This project analyzes customer churn data using Power BI.
It helps identify key factors that contribute to customer churn and provides insights to improve customer retention.

---

## 📁 Files Included

* 📊 Power BI Dashboard (`dashboard.pbix`)
* 📄 Dataset (`Excel file`)
* 📑 Project Documentation (`PDF`)
* 🖼 Dashboard Screenshot

---

## 📊 Key Insights

* Total Customers: **7043**
* Churn Rate: **26.54%**
* Churned Customers: **1869**

### 🔍 Insights:

* Customers with **month-to-month contracts** have higher churn
* **Electronic check** users show more churn
* Customers with **no partner / no dependents** churn more
* Long-term customers (**12+ months**) are more stable
